Quick Start Guide

Get PyBridge running in under 5 minutes

1. Install Dependencies
$ cd pybridge
$ pip install -r requirements.txt
2. Configure

Create a .env file with your API keys, or edit config.json directly: 

# pybridge/.env
ANTHROPIC_API_KEY=sk-ant-...
OPENAI_API_KEY=sk-...
TELEGRAM_BOT_TOKEN=123456:ABC...
HMAC_SECRET=your-random-secret
3. Enable a Channel

Go to Channels and enable at least one:

WhatsApp — Scan QR code (easiest, enabled by default)
Telegram — Create bot via @BotFather
Email — Gmail + App Password
iMessage — macOS only, grant Full Disk Access to Terminal

4. Run
$ python main.py
# or
$ ./start.sh

Send messages from your phone and PyBridge will reply via AI.
